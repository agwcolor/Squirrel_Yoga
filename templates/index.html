What is this really? Squirrel Yoga is a full-stack website Udacity capstone project built using Flask, PostgreSQL, SQLAlchemy ORM, and Auth0 (to add authentication and authorization services), and Heroku (to host the website & PostgreSQL instance). Users can create/modify/delete and manage teachers, courses, events, and tree locations depending on the authorization settings. The website content was inspired by all of the squirrels I saw outside stretching and cavorting during lockdown. Please visit the Github repository to see the code.
